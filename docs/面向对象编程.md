# 面向对象编程思想-OOP
## Java四大特性
### 1 抽象
关键字：抽象建模
### 2 封装
关键字：高内聚，低耦合
换个角度：数据隐藏、黑盒
### 3 继承
关键字：复用，继承树，预防方法污染和方法爆炸（组合或聚合的类关系）
换个角度：扩展已有类
### 4 多态
关键字：覆写override（相同方法不同方法体）

## 类
类是构造对象的模板，由类构造对象的过程称为创建类的实例。

标准的Java库提供几千个类，覆盖范围虽广，但是还是需要自定义Java类，以便描述具体的问题。
既然有标准库，那么自定义Java类就可以参照标准库的设计规范和健壮性考量
